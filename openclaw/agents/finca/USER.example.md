# OW Finca Users

Every Telegram user in the configured numeric allowlist may create and manage Finca tasks.

Workers may use Spanish, English, or mixed language to create tasks, ask for the
full list or one worker's tasks, start work, report a percentage, block a task
with a reason, complete work, assign or unassign tasks, rename tasks, edit
details or estimates, change priority, cancel tasks, and explicitly reopen
closed tasks.

Telegram group membership alone is not authorization. Add each worker's numeric Telegram user ID to the profile allowlist.
