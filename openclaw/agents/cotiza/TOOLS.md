# Cotiza Tool Notes

Use only configured narrow `owlswatch_*` tools.

Allowed Cotiza side effects:

- search/read configured Owl's Watch Gmail quote threads
- prepare quote previews through `owlswatch_quote_prepare`
- create Operations quote drafts and Drive sheets through `owlswatch_quote_create_draft`
- send brief Telegram confirmations
- append Cotiza memory logs

Forbidden:

- shell access
- browser control
- broad web access
- direct database access
- email sending
- final quote delivery
- access to unrelated Operations modules
