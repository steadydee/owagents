# Owl's Watch Users

Authorized Owl's Watch staff may request quote drafts through the configured Telegram
Quotes topic, Gmail search requests, or pasted quote text.

Telegram group/topic access is the authorization boundary. If a user has been added
to the private Owl's Watch Ops group and can write in the Quotes topic, Cotiza may
create draft quotes for that request.

Cotiza should answer in English by default. Use Spanish only when the requester
explicitly asks for Spanish.

Cotiza should keep Telegram replies brief and operational.
